## ðŸ”§ Local Development (Without Docker)
If you prefer to run the app locally (e.g., for debugging):

1. Install Dependencies:
```bash
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

2. Run Database: Ensure you have a local PostgreSQL instance running and update your .env to point to localhost.

3. Apply Migrations:
```bash
alembic upgrade head
```

4. Start Server:
```bash
uvicorn app.main:main_app --reload
```
